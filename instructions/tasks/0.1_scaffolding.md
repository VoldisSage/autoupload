## Description
Create the initial project structure for StreamerTools AutoUploader using Node.js and TypeScript. Set up the basic folders, TypeScript configuration, build scripts and a minimal entrypoint.

## Definition of Done
- A `package.json` exists with scripts:
  - `build` (compiles TypeScript to `dist/`)
  - `start` (runs `node dist/index.js`)
- A `tsconfig.json` exists and compiles all files under `src/` into `dist/`.
- The following folders exist (empty or almost empty):
  - `src/`
  - `config/`
  - `data/`
  - `dev/`
- `src/index.ts` exists and when run via `npm run start` it logs a simple startup message.
- A top-level `README.md` exists explaining:
  - How to install dependencies (`npm install`)
  - How to build (`npm run build`)
  - How to run (`npm run start`)

## Tests
- Run `npm install` and ensure it succeeds.
- Run `npm run build` and verify it finishes without TypeScript errors.
- Run `npm run start` and verify the console prints a message similar to:
  - `StreamerTools AutoUploader started (stub mode)`
