# Task 1.1 – Discord Webhook Logger

## Description
Implement a simple logging service that sends informational and error messages to a Discord channel using a webhook URL from configuration.

## Definition of Done
- A file `src/services/discordLogger.ts` exists with:
  - `async function logInfo(message: string): Promise<void>`
  - `async function logError(message: string, error?: unknown): Promise<void>`
- `logInfo` sends a message to the configured Discord webhook in the format:
  - `✅ <message>`
- `logError` sends a message in the format:
  - `❌ Ha ocurrido un error: <message>\n\`\`\`<error details>\`\`\``
- If the Discord webhook call fails, the service logs an error to the console but does not crash the process.
- The module uses the configuration from `config.ts` (`config.discord.webhookUrl`).

## Tests
- Create a dev script `dev/testDiscordLogger.ts` that:
  - Calls `logInfo("Prueba de info")`
  - Calls `logError("Prueba de error", new Error("Error de prueba"))`
- Add an npm script, for example:
  - `"dev:test-discord": "ts-node dev/testDiscordLogger.ts"` or the equivalent compiled run command.
- Run `npm run dev:test-discord` and verify:
  - Two messages appear in the configured Discord channel with the expected formatting.

## Files to create / modify
